## Ohjelmistotekniikka, harjoitystyö:

###  :snake: Snake game AKA Matopeli  :bug:
Peliin luodaan ensin omat tunnukset, joilla kirjaudutaan sisään. Pelissä kerätään pisteitä nappaamalla palloja ja selviämällä mahdollisimman pitkään elossa. Peli loppuu kun mato kuolee, joka voi olla seuraus joko itseensä osumisesta tai seinään päin menemisestähttps://github.com/johannaval/ot-harjoitustyo/blob/master/dokumentaatio/vaatimusmaarittely.md. Lopuksi peli näyttää top10 listan, jonne vain parhaimmat pisteet keränneet pääsevät.
